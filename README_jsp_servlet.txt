Bước 1: Mờ project bằng eclipse. Chuột phải project -> Show in Local Terminal -> Terminal
Bước 2: Nhập lệnh "mvn clean install" vào Terminal -> Enter
Bước 3: Chuột phải project -> Maven -> Update project
Bước 4: Chạy file create_table.sql và insert_table.sql trong thư mục database
Bước 5: config password mysql trong src/main/resources/db.properties
Bước 6: Run project
